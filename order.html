How an (EDA) Sensor Works

Electrodermal Activity (EDA) sensors monitor skin conductance, which increases with sweating, commonly triggered by stress or seizures.

Skin Conductance Measurement: EDA sensors detect changes in the skin’s conductivity (measured in microsiemens, µS) caused by sweat.

Signal Processing: Data from the sensor is processed to detect patterns associated with seizures.

Alerting: When abnormal conductance levels are detected, alerts are triggered for seizure monitoring.

Key Aspects of (EDA) Sensor Measurements:

  • Baseline Measurement: Records a neutral baseline value during calm states.
  • Detection of Variations: Identifies significant changes from the baseline, indicating possible seizure risk.
  • Alert Thresholds: Algorithms analyze fluctuations to trigger alerts based on abnormal variations.
